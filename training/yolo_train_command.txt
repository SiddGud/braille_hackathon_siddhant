# YOLO Training Command Used

## Command
```bash
yolo task=detect mode=train model=yolov8n.pt data=dataset/data.yaml epochs=100 imgsz=640 batch=16 name=braillevision_run
```

## Hyperparameters
- **Base Model:** yolov8n.pt (YOLOv8 Nano)
- **Epochs:** 100
- **Image Size:** 640
- **Batch Size:** 16
- **Optimizer:** Auto (AdamW)
- **Learning Rate:** Auto (0.01 initial)

## Training Results
Training results, confusion matrix, and metric plots are stored in `training/results/`.
Please see `results/` folder for:
- `results.png` — mAP50 and mAP50-95 curves
- `confusion_matrix.png`
- `val_batch0_pred.jpg` — sample validation predictions
